Datenschutzerklärung

1. Verantwortlicher

Verantwortlich für die Datenverarbeitung auf dieser Website ist der Betreiber gemäß Impressum.

2. Erhobene Daten

Code à Cuisine erhebt keine personenbezogenen Daten im Sinne der DSGVO. Die eingegebenen Zutaten und Präferenzen werden ausschließlich zur Rezeptgenerierung verwendet und nicht dauerhaft mit deiner Person verknüpft.

Zur Steuerung des Tagesquota wird eine anonymisierte, gehashte IP-Adresse temporär gespeichert. Diese wird täglich um Mitternacht automatisch gelöscht.

3. Firebase / Google

Wir nutzen Firebase (Google LLC) zur Speicherung generierter Rezepte. Die Rezeptinhalte werden in der EU (europe-west1) gespeichert. Es werden keine personenbezogenen Daten in der Rezept-Datenbank gespeichert.

4. KI-Generierung

Die Rezeptgenerierung erfolgt über die Anthropic Claude API (Anthropic PBC, USA), vermittelt durch einen n8n-Workflow. Die eingegebenen Zutaten werden zu diesem Zweck an den Dienst übertragen.

5. Deine Rechte

Da keine personenbezogenen Daten gespeichert werden, entfallen Auskunfts-, Lösch- und Berichtigungsansprüche. Bei Fragen wende dich an die im Impressum genannte Kontaktadresse.
