Privacy Policy

1. Data Controller

The data controller for this website is the operator as stated in the Legal Notice.

2. Data Collected

Code à Cuisine does not collect any personal data as defined by the GDPR. The ingredients and preferences you enter are used solely for recipe generation and are not permanently linked to your identity.

To manage the daily usage quota, an anonymized, hashed IP address is stored in Firebase Realtime Database under a date-based key (YYYY-MM-DD). After midnight, a new key is used and previous quota data becomes inactive.

3. Firebase / Google

We use Firebase (Google LLC) to store generated recipes. Recipe data is stored in the EU (europe-west1). No personal data is stored in the recipe database.

4. AI Generation

Recipes are generated via the Anthropic Claude API (Anthropic PBC, USA), routed through an n8n workflow automation. The ingredients you enter are transmitted to this service for that purpose.

5. Your Rights

Since no personal data is stored, rights of access, erasure, and rectification do not apply. If you have any questions, please use the contact address provided in the Legal Notice.
